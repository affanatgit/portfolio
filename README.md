# Syed Shah Rafi Uddin Hussaini - Portfolio

A modern, interactive portfolio website featuring stunning 3D elements and smooth animations. Built with HTML5, CSS3, JavaScript, and Three.js.

## 🌟 Features

- **Modern Design**: Clean, professional layout with a dark theme and neon accents
- **3D Interactive Elements**: Multiple Three.js scenes with animated geometries
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: CSS animations and scroll-triggered effects
- **Interactive Navigation**: Smooth scrolling navigation with mobile hamburger menu
- **Particle Effects**: Dynamic particle systems for visual appeal
- **Hover Effects**: Interactive elements with engaging hover states
- **Performance Optimized**: Efficient rendering and animation management

## 🚀 Live Demo

Open `index.html` in your browser to view the portfolio.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Three.js**: 3D graphics and animations
- **Font Awesome**: Icons
- **Google Fonts**: Inter font family

## 📱 Sections

1. **Hero Section**: Animated introduction with 3D torus knot and particles
2. **About**: Personal information and education with floating cubes
3. **Skills**: Technical skills showcase with rotating spheres
4. **Projects**: Featured projects with unique 3D representations
5. **Leadership**: Event organization experience with network visualization
6. **Contact**: Contact information with communication wave effects

## 🎨 3D Elements

- **Hero**: Animated torus knot with particle field
- **About**: Floating wireframe cubes
- **Skills**: Rotating colored spheres in circular formation
- **Projects**: Custom geometries for each project (cylinder, octahedron)
- **Leadership**: Network nodes with connecting lines
- **Contact**: Pulsating communication waves

## 📋 Setup Instructions

1. **Clone or Download**: Get the portfolio files
2. **Open**: Simply open `index.html` in any modern web browser
3. **Customize**: Edit the content in `index.html` to match your information
4. **Deploy**: Upload to any web hosting service

## 🔧 Customization

### Personal Information
Edit the following in `index.html`:
- Name and title in the hero section
- Contact information
- Education details
- Skills and technologies
- Project descriptions
- Leadership experiences
- Achievements

### Styling
Modify `styles.css` to change:
- Color scheme (currently green/cyan theme)
- Typography
- Layout and spacing
- Animation timings

### 3D Effects
Adjust `script.js` to modify:
- 3D scene configurations
- Animation speeds
- Particle systems
- Interactive effects
